## 商品管理システム

### 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことができます。

商品の新規登録から編集、削除を行うことができ、各商品の在庫を調整することができます。

### 主な機能
- ログイン・ログアウト機能
- 商品一覧画面
- 商品新規登録、編集、削除機能
- 商品検索機能

### 開発環境
PHP 7.4.29
MySQL 10.4.24
Laravel 8.83.27
